| Boundary | Description |
|----------|-------------|
| gold jsonl `drhp_id` → cached record path | The scorer forms `data/redflag/<drhp_id>.json` from a label's `drhp_id`; a non-allow-listed id must not read an arbitrary path |
| committed gold/label files → repo | Hand-authored label + rubric files are developer-authored, not user input; the scorer trusts their shape only after parse |
| cached prediction JSON → scorer | Predictions are read from disk and compared; malformed JSON must fail loudly, not silently mis-score |
